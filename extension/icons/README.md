# Icon Placeholder

The extension requires icons in PNG format (16x16, 48x48, 128x128).

You can:
1. Convert the included `icon128.svg` to PNG using an online tool
2. Or create your own icons

Place the files as:
- `icons/icon16.png` (16x16 pixels)
- `icons/icon48.png` (48x48 pixels)  
- `icons/icon128.png` (128x128 pixels)

For now, you can use any placeholder images to test the extension.
